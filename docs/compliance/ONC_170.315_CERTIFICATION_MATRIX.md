## How To Read This Matrix

This is an **internal pre-assessment evidence matrix**. It is the crosswalk a certifying ACB (Drummond Group) uses to locate evidence for each criterion. **It does not itself constitute certification** — certification is granted only after ACB-witnessed testing against the official ONC test procedures.

| Status | Meaning |
|--------|---------|
| **READY** | Implemented, tested, and verified in source on the date above. Ready for ACB test procedure. |
| **READY (visual pending)** | Code + tests complete and live-proven; awaiting final UI visual acceptance documentation. |
| **GAP — buildable** | Not yet certification-ready; internal work scoped, no external dependency. |
| **BLOCKED — external** | Cannot be completed without a third-party vendor agreement (timeline outside our control). |

**Evidence convention:** Each row cites the primary source file(s) and, where applicable, the test file and route. Paths are repo-relative. "Route" means the in-app reachable entry point (a real nav card or link, not a typed URL), per the codebase "DONE MEANS DONE" rule.
