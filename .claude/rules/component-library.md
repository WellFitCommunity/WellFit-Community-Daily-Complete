# Component Library Reference

Before using any Envision Atlus (EA) design system component, you MUST read the component reference:

**`docs/AI_COMPONENT_REFERENCE.md`**

This contains the exact valid variants for EABadge, EAAlert, and other EA components. Do NOT guess variant names — they follow medical severity standards, not generic UI conventions (e.g., use `'critical'` not `'danger'`, use `'normal'` not `'success'` for EABadge).

Failure to check this file before writing EA component code has caused repeated rework across multiple sessions.
